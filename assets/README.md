# Brand assets

| File | Use |
|------|-----|
| `logo.svg` | the mark alone (square) — avatars, embeds |
| `logo-wordmark.svg` | mark + wordmark, blue text (light backgrounds) |
| `logo-wordmark-dark.svg` | mark + wordmark, light text (dark backgrounds) |
| `social-card.svg` / `social-card.png` | 1280×640 GitHub social preview |
| `favicon.svg` | app-icon style mark (dark rounded tile) |
| `favicon/favicon.ico` | multi-size icon (16/32/48) for the web |
| `favicon/favicon-{16,32,48,180,192,512}.png` | individual favicon / touch-icon sizes |

Brand colors: gradient `#5B8DEF` → `#22D3EE`; wordmark blue `#2563EB`; dark surface `#0B1220`.

The PNGs are rendered from the SVGs; edit the SVG and re-export to change them.
